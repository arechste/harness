# My Life

The principal's life-management hub. Five flat concept folders (one file per concept, no nesting):

- **Topics/** — recurring subjects of interest, study, or attention
- **Habits/** — behaviors the principal is building, maintaining, or dropping
- **Goals/** — outcomes the principal is working toward (paired with `PKM/Goals.md` for the headline goals)
- **Projects/** — concrete work the principal is doing (personal projects, not the team's harness work)
- **Key Elements/** — the principal's foundational values, principles, and identifiers (the things that anchor everything else)

## Conventions

- One file per concept, kebab-case (e.g., `home-network.md`, `morning-routine.md`).
- Cross-link with `[[wikilinks]]` — Journal entries reference Topics/People/Projects, never duplicate facts.
- SSOT: if a fact about a topic appears in two places, the file in the relevant concept folder is the source; the Journal entry links to it.
- Empty for now — the principal seeds content as it becomes relevant; the team (QUILL writes + TOWER librarian-passes) helps with capture and connective tissue, but the principal owns final content.
